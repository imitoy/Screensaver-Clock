# Screensaver Clock

A simple and elegant screensaver clock application built with Flutter, which generated by Gemini.

## Features

- **Digital Clock:** Displays the current time in HH:mm:ss format.
- **Fullscreen Mode:** Immerse yourself in the clock display without distractions.
- **Resizable Font:** Adjust the font size of the clock to your liking.
- **Screen Rotation:** Rotate the screen to either portrait or landscape mode.
- **Settings Page:** A dedicated page to manage your preferences.

## Getting Started

To get started with the app, follow these steps:

1. **Installation:**
   - Clone this repository to your local machine.
   - Run `flutter pub get` to install the required dependencies.
2. **Running the App:**
   - Run `flutter run` to start the application on your connected device or emulator.
3. **Usage:**
   - The main screen displays the digital clock.
   - Use the buttons on the title bar to:
     - Toggle fullscreen mode.
     - Rotate the screen.
     - Navigate to the settings page.
   - In fullscreen mode, tap the screen to exit.
   - On the settings page, use the slider to adjust the font size of the clock.

Enjoy the simple and elegant screensaver clock!
